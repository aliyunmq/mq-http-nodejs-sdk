<async> consumeHalfMessage(numOfMessages, waitSeconds)

消费检查事务半消息,默认如果该条消息没有被 {commit} 或者 {rollback} 在NextConsumeTime时会再次消费到该条消息

Parameters:
Name Type Description
numOfMessages int

每次从服务端消费条消息
waitSeconds int

长轮询的等待时间（可空），如果服务端没有消息请求会在该时间之后返回等于请求阻塞在服务端，如果期间有消息立刻返回
